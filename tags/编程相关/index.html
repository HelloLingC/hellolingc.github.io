读书笔记 - Introduction to Computer Organization: ARM Assembly Language Using the Raspberry Pi

今天读的书是：Introduction to Computer Organization: ARM Assembly Language Using the Raspberry Pi by Robert G. Plantz 1.3 How the Subsystems Interact 关于三个subsystem的交互方式：CPU，Memory，I/O。 以及介绍三种线路：address bus，data bus，control bus 由于bus的占用，CPU无法在指定某一条指令同时去读取下一条instruction。Harvard architecture 由于bus的速度跟不上现代cpu的执行速度，所以就会出现von Neumann bottleneck(冯诺依曼瓶颈)。因而cpu都会加入cache memory。
